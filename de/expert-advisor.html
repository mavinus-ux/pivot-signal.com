Section 1 · The Basics

Was ist ein Expert Advisor?

Automated trading programs that execute your strategy on the markets — without emotions, without screen time, without missed setups.

An Expert Advisor (EA) is a software program that runs inside the MetaTrader 5 trading platform. It's written in MQL5 (MetaQuotes Language 5) and follows a strict set of rules you define in advance — entry conditions, position sizing, stop-loss, take-profit, and exit logic.

Once you attach an EA to a chart, it scans the market 24/5, identifies setups that match its rules, and executes trades automatically. No emotions, no hesitation, no "I'll close at lunch and re-open later". Pure discipline, repeated thousands of times.

Think of it as a trading robot that follows your exact playbook — but faster, more consistent, and without the human flaws that cost most traders money.

  • Algorithmic precision — executes the same logic on every setup
  • 24/5 uptime — never sleeps, never takes a break, never misses a setup
  • Backtested rules — every entry/exit is based on tested strategy logic
  • Emotion-free — no fear, no greed, no revenge trading
  • Multi-pair capable — trade EUR/USD, GBP/JPY, XAU/USD simultaneously
  • Full control — adjust risk, lot size, hours, instruments anytime

Section 2 · Implementation

EA installieren in MetaTrader 5

From download to live trading in about 5 minutes. No coding required.

1

Download the EA file

After purchase, you receive a .ex5 or .mq5 file via email. Save it somewhere you can find it.

2

Open MT5 & Data Folder

In MetaTrader 5: File → Open Data Folder. Then navigate to:

MQL5 → Experts
3

Paste the EA file

Drop the .ex5 file into the Experts folder. Restart MT5 so it loads.

4

Enable AutoTrading

In the MT5 toolbar, click the AlgoTrading button so it shows green. This is mandatory.

5

Drag EA onto chart

From the Navigator → Expert Advisors panel, drag your EA onto any chart (e.g. EUR/USD M15).

6

Configure & test

Open the EA settings. Set your lot size, risk %, max drawdown. Run on demo first for 2–4 weeks.

⚠️ Important: Always test an EA on a demo account for at least 2–4 weeks before going live. Every market cycle is different, and the EA might need parameter tuning for your broker's spreads, execution speed, and trading conditions.
